TENTANG SAYA

Halo semua, perkenalkan, Saya Vernanda Mulia Hamonangan Manurung, atau biasa dipanggil Veri oleh teman-teman terdekat. Saya berasal dari Surabya. Saya adalah seseorang yang memiliki dua minat utama dalam hidup, seni lukis dan dunia teknologi. Sejak kecil, saya sudah terpesona oleh keindahan warna dan bentuk. Lukisan bagi saya adalah cara untuk mengungkapkan perasaan dan ide dalam bentuk visual. Tiap goresan kuas adalah cerita yang ingin saya bagikan kepada dunia. Namun, selain dunia seni, saya juga memiliki semangat yang besar dalam olahraga. Bermain basket dan badminton adalah hobi yang membuat saya merasa hidup dan penuh semangat. Saat ini, saya sedang menempuh pendidikan di PENS dan mengambil jurusan Teknik Informatika. Saya memiliki impian besar untuk menjadi seorang programmer yang handal. Bagi saya, pemrograman adalah seperti menciptakan karya seni dalam bentuk kode. Saya ingin menjadi seorang fullstack developer yang tidak hanya menguasai aspek-aspek teknis, tetapi juga memiliki pandangan estetika yang bisa menghadirkan keindahan dalam setiap solusi teknologi yang saya ciptakan.

Dengan semangat yang tak pernah padam, saya berusaha menggali ilmu sebanyak-banyaknya di kampus dan berusaha mewujudkan impian saya untuk menjadi seorang fullstack developer yang bisa memberikan kontribusi nyata dalam dunia teknologi. Jadi, itulah sedikit gambaran tentang diri saya, saya berharap bisa terus mengembangkan diri untuk mencapai tujuan dan impian saya. Terima kasih!

SKILL

HTML

Saya memiliki keterampilan dalam HTML, yang memungkinkan saya untuk membuat halaman-halaman web. HTML adalah bahasa yang digunakan untuk membuat struktur dan konten di situs web. Dengan keahlian ini, saya bisa menambahkan teks, gambar, tautan, dan elemen lainnya ke halaman web, sehingga membuatnya terlihat dan berfungsi dengan baik. Ini memungkinkan saya untuk berkontribusi dalam pengembangan dan perancangan situs web.

CSS

Saya memiliki keterampilan dalam CSS, yang memungkinkan saya untuk mengatur tampilan halaman-halaman web. Saya dapat menggambarkan elemen-elemen HTML dengan berbagai gaya, sehingga halaman web menjadi estetis dan mudah dibaca.

C CODE

Dengan keahlian ini, saya bisa membuat program komputer yang melakukan tugas-tugas tertentu, seperti mengelola data, dan berinteraksi dengan perangkat keras.
